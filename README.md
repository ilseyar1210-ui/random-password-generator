**Автор:** Белотдинова Милана Дамировна

## Описание

Random Password Generator — это графическое приложение на Python (Tkinter) для генерации случайных паролей. Позволяет настраивать длину пароля и используемые символы, сохраняет историю сгенерированных паролей в JSON.

## Установка и запуск

1. Убедитесь, что установлен Python 3.7+
2. Скачайте файл `main.py` из репозитория
3. Запустите в терминале:
   ```bash
   python main.py
